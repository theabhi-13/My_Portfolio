# My Portfolio

Personal portfolio site built with Vite + React + TypeScript + Tailwind CSS.

> This repository contains a fast, accessible, responsive single-page portfolio showcasing projects, skills, experience and contact information.

---


## Demo

- Live: https://your-website.example.com

---

## Features

- Responsive, mobile-first layout
- Smooth animations with `framer-motion`
- Accessible components using Radix UI and shadcn-style patterns
- Sections: Hero, About, Skills, Experience, Projects, Education, Contact

---

## Project structure (short)

- `src/` — source files
- `public/` — static assets
- `screenshots/` — place screenshots here (used by README)
